Gracedemo

Gr@ce Network - demo

Download this project as a .zip file Download this project as a tar.gz file

Ce prototype de portail de démonstration 100% open source, nommé grace demo, a été développé par Makina Corpus en collaboration avec CADaGEO et Oise THD. L'objectif est de proposer un outil permettant d'exploiter une base de données conforme au geostandard ANT.

Vous pouvez y accéder en vous rendant ici : grace.makina-corpus.net

Gr@ce est le nom du portail de géoréférencement et de recensement des infrastructures de communications électroniques de la région Aquitaine. http://grace.aquitaine.fr/accueil/

Le modèle de données dédié à l'aménagement numérique du territoire (ANT) est disponible sous licence libre et est en grande partie à l'origine du géostandard ANT validé par la Covadis. Toutefois Grace reste un nom d'usage pour désigner ce modèle de données. Page officielle : http://www.certu.fr/geostandard-amenagement-numerique-a350.html

Les données réseau sont la propriété de Gironde Numérique et ont été aimablement mises à disposition de CADaGEO afin de promouvoir le projet Gr@ce.

Un groupe de discussion ouvert à tous pour traiter des sujets liés à Grace est disponible. https://groups.google.com/forum/#!forum/geostandard-ant-users geostandard-ant-users@googlegroups.com
